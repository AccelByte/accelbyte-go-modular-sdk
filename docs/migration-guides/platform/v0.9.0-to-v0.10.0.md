# [v0.10.0]

## BREAKING CHANGE

### Models

The following model(s) has been updated.

- Model `BinarySchema` has been removed from `platformclientmodels`. All references to
  `platformclientmodels.BinarySchema` must be replaced with `io.Writer`.

### Operations

The following operation(s) has been updated.

- Operation `PublicGetQRCode`
    - A required `writer io.Writer` parameter has been added to the method. The response payload type has also
      changed from `*platformclientmodels.BinarySchema` to `io.Writer`.

      ```diff
      - func (aaa *PaymentStationService) PublicGetQRCodeShort(input *payment_station.PublicGetQRCodeParams) (*payment_station.PublicGetQRCodeResponse, error)
      + func (aaa *PaymentStationService) PublicGetQRCodeShort(input *payment_station.PublicGetQRCodeParams, writer io.Writer) (*payment_station.PublicGetQRCodeResponse, error)
      ```

      The `Unpack()` return type and the `Data`/`Payload` fields on the response structs have changed
      accordingly:

      ```diff
      - func (m *PublicGetQRCodeResponse) Unpack() (*platformclientmodels.BinarySchema, *platformclientmodels.ApiError)
      + func (m *PublicGetQRCodeResponse) Unpack() (io.Writer, *platformclientmodels.ApiError)
      ```

- Operation `CheckBalance`
    - Return type changed. The operation now returns the response payload alongside the error instead of
      returning only an error.

      ```diff
      - func (aaa *WalletService) CheckBalanceShort(input *wallet.CheckBalanceParams) error
      + func (aaa *WalletService) CheckBalanceShort(input *wallet.CheckBalanceParams) (*wallet.CheckBalanceResponse, error)
      ```

      The `Unpack()` method on `CheckBalanceResponse` has changed accordingly:

      ```diff
      - func (m *CheckBalanceResponse) Unpack() *platformclientmodels.ApiError
      + func (m *CheckBalanceResponse) Unpack() (*platformclientmodels.CheckBalanceResponse, *platformclientmodels.ApiError)
      ```
